# 🎮 Game Assets

This folder contains assets for Penkey mini-games.

## Folder Structure

- **dice_roll/** - Dice images and backgrounds
- **duck_memory/** - Colored duck images for memory game
- **monkey_penguin/** - Monkey and penguin character sprites
- **cup_stack/** - Coffee cup images and stack backgrounds
- **donut_catcher/** - Food item images (donuts, pastries, etc.)

## Image Guidelines

- **Format:** PNG with transparency preferred
- **Size:** 256x256px for game pieces, 512x512px for backgrounds
- **Style:** Playful, colorful, matching Penkey brand
- **Naming:** Use lowercase with underscores (e.g., `yellow_duck.png`)
