# 📱 Staff QR Scanner - Quick Reference Guide

## 🎯 What Can You Scan?

### 1️⃣ Customer Profile QR Codes
**Format:** `PROFILE-xxxxx`  
**Where:** Customer's app profile page  
**Use For:**
- Daily check-ins
- Adding coffee stamps
- Awarding bonus beans

### 2️⃣ Reward QR Codes
**Format:** `REWARD-xxxxx` or `COFFEE-xxxxx`  
**Where:** Customer's rewards page in app  
**Use For:**
- Redeeming free coffee
- Redeeming earned rewards
- Claiming prizes

---

## 🔍 How to Use

### **Option 1: Camera Scanner (Recommended)**
1. Click **"Open Camera Scanner"** button
2. Point camera at customer's QR code
3. Wait for auto-detection (1-2 seconds)
4. Follow on-screen prompts

### **Option 2: Manual Entry**
1. Ask customer for their QR code
2. Type code in input field
3. Click **"Scan"** button
4. Follow on-screen prompts

---

## 📋 Common Tasks

### ✅ Daily Check-In
```
1. Scan customer PROFILE QR code
2. Customer info appears
3. Click "Check-in" button
4. Customer earns beans!
```

### ☕ Add Coffee Stamp
```
1. Scan customer PROFILE QR code
2. Customer info appears
3. Click "Add Coffee Stamp" button
4. Stamp added (10 stamps = free coffee!)
```

### 🎁 Redeem Reward
```
1. Scan customer REWARD QR code
2. Confirmation dialog appears
3. Verify customer name
4. Click "OK" to confirm
5. Reward redeemed!
```

---

## ⚠️ Error Messages

| Message | Meaning | What to Do |
|---------|---------|------------|
| "Customer not found" | Invalid profile QR | Ask customer to refresh their QR code |
| "Already checked in today" | Customer checked in earlier | They can check in again tomorrow |
| "Reward not found" | Invalid reward QR | Verify QR code is correct |
| "Already redeemed" | Reward was used | Customer already claimed this reward |
| "Reward has expired" | Reward validity ended | Customer needs to earn new reward |

---

## 🎯 Quick Tips

### ✅ DO:
- ✅ Verify customer name before confirming redemptions
- ✅ Use camera scanner for speed
- ✅ Check customer has correct QR code visible
- ✅ Wait for confirmation message

### ❌ DON'T:
- ❌ Redeem rewards without confirmation
- ❌ Scan QR codes from screenshots (may be old)
- ❌ Process expired rewards
- ❌ Skip verification steps

---

## 🆘 Troubleshooting

### Camera Won't Open?
1. Check browser permissions
2. Allow camera access
3. Refresh page
4. Use manual entry as backup

### QR Code Won't Scan?
1. Ensure good lighting
2. Hold phone steady
3. Keep QR code in frame
4. Try manual entry instead

### Wrong Customer Loaded?
1. Click "Reset" button
2. Scan correct QR code
3. Verify customer name

---

## 📞 Need Help?

1. Check error message on screen
2. Try manual entry if camera fails
3. Ask customer to refresh their QR code
4. Contact support if issue persists
