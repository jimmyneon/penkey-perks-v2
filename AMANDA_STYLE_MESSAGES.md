## 🎯 GPS Features

### **QR Scanner Button:**
- **Only appears when near Penkey** (within 100m)
- Pulses/animates to draw attention
- Disappears when far away
- Makes scanning feel special!

### **Message Changes:**
- **Far away:** General encouragement
- **Nearby:** "OMG you're so close!"
- **At Penkey:** "YAAAS you're here!"

## 🎨 Message Rotation

Messages rotate daily based on day of week:
- **Monday:** Message 1
- **Tuesday:** Message 2
- **Wednesday:** Message 3
- etc.

Keeps content fresh and gives customers something new each visit!

---

## 📱 Location Detection

### **How It Works:**
1. App checks GPS on dashboard load
2. Calculates distance to Penkey
3. Shows different messages based on distance
4. QR scanner appears when nearby

### **Distance Zones:**
- **> 100m:** Regular messages
- **< 100m:** "You're nearby!" messages
- **At Penkey:** "You're here!" messages
